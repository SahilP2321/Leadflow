Project Reflection

Overview
I have successfully built a complete end-to-end lead intake and follow-up automation system that processes prospect information, enriches company data from public sources, generates personalized audit reports in PDF format, and delivers them via email - all without human intervention.

What I Built
The system consists of:

Lead Intake Form - Captures prospect information (name, email, company, position)

Company Enricher - Fetches data from Wikipedia and web scraping to gather company details including industry, founding year, headquarters, employee count

PDF Report Generator - Creates professional 12-page strategic audit reports using ReportLab

Email Service - Sends personalized emails with PDF attachments via Gmail SMTP

Async Processing - Uses Celery and Redis for background task processing

Real-time Updates - HTMX provides live status updates without page refresh

Challenges Overcome
PDF Formatting Issues

Problem: HTML tags appearing in PDF

Solution: Used ReportLab's native Paragraph objects instead of raw HTML

Celery Connection Errors

Problem: Celery trying to connect to RabbitMQ instead of Redis

Solution: Explicitly set CELERY_BROKER_URL in settings

Email Delivery

Problem: Emails not being received

Solution: Configured Gmail App Password instead of regular password

Import Errors

Problem: Relative import issues in urls.py

Solution: Changed from from ..lead_automation.leads import views to from . import views
